## Project Overview

The **Social Media Analytics Teaching Toolkit** is a comprehensive resource designed to empower faculty at the UCF Rosen College of Hospitality Management. This toolkit facilitates the integration of live data analytics into the hospitality curriculum, allowing students to engage with real-world market trends using tools like Google Trends and Social Blade.

This project addresses the need for up-to-date, improved curriculum materials that bridge the gap between theoretical concepts and practical, data-driven decision-making in the hospitality industry.

## Project Status

**Current Phase: Release 1.0 (Deployed)**
- Core functionality implemented.
- Templates fully integrated.
- Student guides and faculty resources available.

## Key Features

*   **Turnkey Module Integration:** Pre-built templates aligned with HFT course learning outcomes.
*   **Live Data Focus:** Replaces static case studies with real-time market data analysis.
*   **Student & Faculty Guides:** Comprehensive documentation to support both teaching and learning.
*   **Automated Reporting:** Streamlined generation of PDF reports and PPTX presentations for grading.
*   **Zero Cost:** Utilizes free, public data sources, eliminating the need for expensive software or textbooks.

## Technology Stack

This project is built using modern web technologies to ensure a responsive and interactive user experience:

*   **Frontend Framework:** React (via Vite)
*   **Styling:** Tailwind CSS (with customization for UCF branding)
*   **Routing:** React Router DOM
*   **Animations:** Framer Motion
*   **Icons:** Lucide React

## Getting Started

To run this project locally for development or review:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hancer70/technology-fellowship.git
    cd technology-fellowship
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
